# pokedex

## Descrição do Projeto
<p align="center">Este projeto foi realizado para o desafio BlincTek Academy com intuito de avaliar minhas habilidades como programadora. O desafio consiste na listagem de pokémons através da PokéAPI que pode ser acessada por esse link: https://pokeapi.co/
  Para o desafio era  necessário realizar a listagem de todos 100 primeiros pokémons pelo consumo do endpoint https://pokeapi.co/api/v2/pokemon?limit=100. Os atributos que deveriam ser listados de cada Pokémon são:
1. **Nome**
2. **id**
3. **URL da Foto**
</p>
